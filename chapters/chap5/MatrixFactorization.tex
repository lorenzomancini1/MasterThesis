\chapter{Matrix Factorization}\label{chap:fact}
As we have pointed out multiple times, the power of the \acrshort{hm} relies on the ability to retrieve a corrupted version of a pattern and this can be done for all the configurations stored by the system. In this chapter we focus on the \acrlong{shm} where the storage procedure happens through the computation of the coupling matrix $J$. Any configuration \bsigma which is similar to one of the stored patterns, but not equal, can be brought back to the original pattern by minimizing energy~\cref{eq:energy_ising}. In~\cref{chap:radius} we have seen that each pattern define its own \acrlong{ba} which has radius dependent on the load parameter.

Now, we face the problem of energy minimization from a differen perspective. Let's say that we are provided with a coupling matrix $J$ generated by $M$ unknown patterns \bxi, with $\mu = 1, \dots, M$. The question is:
is it possible to infer the patterns that generated such matric $J$?
\begin{equation*}
    J_{N \times N} \overset{?}{\rightarrow} \bxi.
\end{equation*}
At first glance this might seem a very hard problem and indeed here we present some unsuccessful numerical results.\\
We refer to this problem as a \emph{matrix factorization} one since, if we call $\xi$ the $N \times M$ matrix where each column is a stored pattern, then from~\cref{eq:weights}:
\begin{equation}\label{eq:factorization}
    J = \frac{1}{N} \cdot \xi \xi^T,
\end{equation}
that is, we have decomposed $J$ into a product of two matrices. However, in~\cref{eq:factorization}, one must be careful to set all the diagonal elements to $0$ to keep consitency with the Hebbian rule.

As a starting point, we consider the region of the phase diagram related to the \acrlong{shm} where patterns are global minima. This translates into considering the load parameter $\alpha \lesssim 0.05$. At this point, the goal is to find the global minima of the energy defined by $J$. Of course, in the algorithms that we are going to show, we act as if we know the matrix of the patterns $\xi$, and indeed we use the latter to check performances. However, it must be stressed that this is an unsupervised learning task, therefore, in the ideal situation, we do not know $\xi$, thus forcing us to explore some other methods to investigate the behaviour of the algorithms. A solution to this would be, for example, to compare the energy of the configuration obtained with the theoretical energy of a global minima, which is given by the replica trick analysis (chiedere prof).
